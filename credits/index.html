Credits

MIYO keeps its upstream roots visible.

This project is a rebranded continuation of Usagi, and Usagi is derived from Kotatsu. That lineage belongs in the public docs, not buried in commit history.

Upstream

Kotatsu

Kotatsu established much of the reader and parser ecosystem that MIYO benefits from. MIYO does not host Kotatsu content or alter third-party hosted manga links from this website.

View Kotatsu on GitHub

Fork lineage

Usagi

MIYO is a rebranded fork and continuation of Usagi, with additional branding, release automation, performance work, download changes, and refinement infrastructure.

View Usagi on GitHub

MIYO-specific work

  • Rebranded package and release metadata for org.koharu.miyo.
  • GitHub Actions release workflow and model artifact publishing.
  • Download reliability, local reading, and storage validation work.
  • C++ helper and refinement infrastructure.
  • GitHub Pages website, release links, and public docs.

Content boundary

This website only describes MIYO and links to GitHub Releases. It does not host manga, provider catalogs, bypass services, or third-party source content.
