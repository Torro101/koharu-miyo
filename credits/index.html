Credits

MIYO carries visible credit for its upstream roots.

This project is a rebranded continuation of Usagi, and Usagi is derived from Kotatsu. The website keeps that lineage visible.

Kotatsu

Kotatsu established much of the reader and parser ecosystem lineage that MIYO benefits from. MIYO does not host Kotatsu content or change Kotatsu-hosted manga links from this website.

Usagi

MIYO is a rebranded fork and continuation of Usagi, with additional app branding, performance work, download changes, refinement work, and release infrastructure.

View Usagi on GitHub

MIYO-specific work

  • Rebranded package and release metadata for `org.koharu.miyo`.
  • GitHub Actions release workflow and model artifact publishing.
  • Download reliability and local reading improvements.
  • Native helper and refinement infrastructure.

Content boundary

This website only describes MIYO and links to its GitHub Releases. It does not host manga, source catalogs, or third-party provider content.
